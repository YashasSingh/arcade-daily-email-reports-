Using Task Scheduler (Windows):

--------------------------------------------------------

Open Task Scheduler.

Create a new Basic Task.

Follow the wizard to schedule the task daily and point it to the Python executable and your script, e.g.:


//////////////////////////////////////////////////////
sh
-----
C:\Python39\python.exe C:\path\to\your\script.py
/////////////////////////////////////////////////////
